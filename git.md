# GIT (cli + remote)

## GIT cli (CommandLineInterface)

`cd path/to/folder` navigate to your folder  
`git init` initializes / creates a **local repository**

> ❗️ do **not initialize** / create a git repository **inside another** git repository!  
> Check: `git status`
